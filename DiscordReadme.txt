    sudo pip3 install py-cord==2.0.0rc1

Get Discord Bot Token and other Settings with Discord API:

1.  Open User Settings in Discord Client
2.  Navigate to Advanced under App Settings
3.  Turn on Developer Mode and click on the Discord API Link
4.  Login if Asked to do so
5.  Create New Application
6.  give it a name
7.  Click on the bot menu on the left
8.  Click the Add Bot button and confirm your action
9.  Click Reset Token, confirm and enter your 2FA code if asked
10. Copy the Token and paste it into token.txt
11. Scroll down to Message Content Intent and Enable
